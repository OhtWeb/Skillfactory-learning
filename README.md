# Парсер киноафиши (Kinoafisha Parser)

Программа для автоматизированного сбора данных о киносеансах с портала Kinoafisha.info (https://www.kinoafisha.info/).

 1. Постановка задачи и обоснование
 Задача:
Разработать инструмент, который позволяет пользователю быстро получить актуальный список фильмов, идущих в кинотеатрах конкретного города в выбранную дату, с возможностью сортировки и выгрузки данных в структурированные форматы.

 Обоснование:
Поиск информации на веб-сайтах вручную может быть неудобен из-за обилия рекламы и необходимости переключаться между вкладками для сравнения рейтингов. Данный скрипт автоматизирует процесс (Web Scraping), позволяя:
- Выбирать город из списка поддерживаемых (Москва, Санкт-Петербург, Новосибирск).
- Указывать произвольную дату.
- Настраивать критерии сортировки (рейтинг, название, популярность).
- Сохранять результаты в форматы CSV и Excel для дальнейшего анализа или личного использования.

---

 2. Стек технологий
- Язык: Python 3.x
- Библиотеки:
  - requests: отправка HTTP-запросов к серверу.
  - BeautifulSoup4 (bs4): парсинг HTML-кода страницы.
  - pandas: обработка данных и экспорт в таблицы.
  - lxml: быстрый парсер для BeautifulSoup.
  - openpyxl: движок для создания файлов .xlsx.

---

 3. Инструкция по сборке и запуску

 Предварительные требования
Убедитесь, что у вас установлен Python версии 3.7 или выше.

 Шаг 1: Клонирование репозитория или создание папки
Скопируйте файл со скриптом (например, main.py) в рабочую директорию.

 Шаг 2: Установка зависимостей
Откройте терминал (командную строку) и установите необходимые библиотеки одной командой:
pip install requests beautifulsoup4 pandas lxml openpyxl


 Шаг 3: Запуск программы
Запустите скрипт через интерпретатор Python:
python main.py


---

 4. Порядок работы с программой
После запуска следуйте подсказкам в консоли:
1. Выбор города: Введите цифру 1, 2 или 3.
2. Ввод даты: Укажите интересующую дату в формате дд.мм.гггг (например, 25.10.2023).
3. Выбор сортировки: Определите порядок отображения фильмов.

 Результат выполнения:
- В консоль будут выведены первые 5 фильмов согласно выбранным фильтрам.
- В корневой папке проекта будут созданы два файла:
    - Movies.csv (текстовый формат с разделителями-запятыми).
    - Movies.xlsx (формат таблицы Excel).

---

 5. Особенности реализации
- User-Agent: Программа имитирует запрос от реального браузера (Chrome), чтобы избежать блокировок со стороны сервера.
- Обработка данных: Скрипт автоматически разделяет объединенные данные о стране и дате выхода, предоставляя чистую таблицу.
- Валидация: Реализована обработка ошибок ввода даты через try-except и подмена некорректного выбора города значением по умолчанию (Москва).
